Flexible Layout

Welcome, this page has a flexible layout utilizing built-in CSS flex, flex-direction and the float property. When the flex-direction is set to column and the float property is set to left, the two main containers or content areas will automatically sit nicely next to each other. When displayed on a desktop browser the nav bar sits on the left-hand side of the screen, and when viewed on smaller screen or mobile devices the nav bar is situated at the top of the page. Resize the browser to play with this effect.

Sticky Nav Bar

The sticky Nav bar is an easy way to enable a user to at any point in the page select a seperate link as opposed to scrolling all the way to the top of the page. Sticky nav effect can be easily done by setting the position property to fixed or sticky ex: (position: fixed;). A nice feature would be giving the user the option to disable the sticky effect on/off.

Mobile Friendly

Mobile friendly goes hand-in-hand with having a flexible layout. If you want your site to be successful a mobile friendly version is a must. To accomodate mobile devices a media query was set to 700px, which accomodates most moble devices. Many sites i've browsed use mulitple media queries to accomodate more devices and tailor the page accordingly.

CSS Classes

CSS Classes were an absolute must in order to manipulate each individual section. Classes make styling html a breeze although, at times it can be frustrating. Example class (class="my-class") this is the property used in the html code. Then, in CSS it is accessed (my-class { }.)

CSS ID's

CSS ID's allow you to style your html elements as well as link certain parts of a page. For example by clicking on the nav bar icon or the features themselves will scroll the page content to the id selected. Go ahead give it a try.

Border Radius

Border radius add some extra style to components. Notice the main content area has a radius on it. Also, the links have a border radius when they are hovered over with the mouse pointer.

Hover Effect

The hover effect can be done in CSS to give the user a visual cue that something is selectable. Example hover effect (a:hover { background-color: white; }).

Media Query

Media Query(s) are as touched on in the mobile friendly section essential to accomodate different sized devices. This page used a media query at a 700 pixel breakpoint to allow the contents to be rearranged for better viewing.

Shadow

Shadows, a simple but effecting styling effect to create depth on a page. This page uses shadows on the main content area and just below the navigation bar..did you notice it?
